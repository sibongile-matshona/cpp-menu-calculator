# C++ Menu Calculator

This program allows the user to compute the sum, mean, minimum, and maximum of a set of numbers using a menu-driven interface.

This project was completed as part of a Computer Engineering course at the University of North Texas.

## Features
- Menu system using switch statements
- User input handling
- Loop-based calculations
- Input validation

## What I learned
- How to use loops and conditionals together
- How to structure a program with a menu system
- How to handle user input in C++

## How to run
Compile and run using a C++ compiler:

g++ menu_calculator.cpp  
./a.out
